# CLAUDE.md

## Project Overview

Discover Amsterdam — a city travel guide website built with Astro 5, Tailwind CSS 4, and deployed to Cloudflare Pages.

## Commands

```bash
npm run dev       # Start dev server at localhost:4321
npm run build     # Production build to ./dist/
npm run preview   # Preview production build locally
```

## Content Rules

- All prices in EUR and USD
- 11 Amsterdam destinations/neighborhoods: Centrum, Jordaan, De Pijp, Museum Quarter, Amsterdam North, Vondelpark Area, Red Light District, Plantage, Haarlem, Zaanse Schans, Keukenhof
- 7 regions: centrum, canal-ring, east, west, north, south, day-trip
